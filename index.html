HTML Basics

HTML (Hypertext Markup Language) is the standard markup language for creating web pages.

Here are some basic elements:

  • <!DOCTYPE html>: Defines the document type and HTML version.
  • <html>: Root element of an HTML page.
  • <head>: Contains meta-information about the document.
  • <title>: Sets the title of the document (displayed in the browser's title bar or tab).
  • <body>: Contains the visible content of the document.

Text Elements

HTML provides various tags to structure and format text:

  • <h1> to <h6>: Headings (from most important <h1> to least important <h6>).
  • <p>: Paragraph.
  • <em>: Emphasized text (usually rendered in italics).
  • <strong>: Important text (usually rendered in bold).
  • <span>: Inline container for stylistic purposes.

Structural Elements

Elements for structuring the layout of your page:

  • <div>: Generic container for grouping elements.
  • <header>, <footer>, <main>, <section>: Semantic elements for different parts of a webpage.
  • <nav>: Navigation links.
  • <article>: Independent, self-contained content.
  • <aside>: Content aside from the main content (like sidebars).

Forms

Elements used to create forms for user input:

  • <form>: Container for form elements.
  • <input>: Input control (text, checkbox, radio buttons, etc.).
  • <textarea>: Multi-line text input.
  • <select>: Dropdown list.
  • <button>: Button.

Media

Tags for embedding media files:

  • <img>: Embeds an image.
  • <audio>: Embeds audio content.
  • <video>: Embeds video content.
  • <iframe>: Embeds an external webpage.

Tables

Tags for creating tables:

  • <table>: Defines a table.
  • <tr>: Defines a row in a table.
  • <th>: Defines a header cell in a table.
  • <td>: Defines a cell in a table.
  • <caption>: Defines a table caption.

HTML5

New features introduced in HTML5:

  • More semantic elements like <header>, <footer>, <nav>, etc.
  • Form enhancements such as <input type="date">, <input type="email">, etc.
  • Media elements like <audio> and <video>.
  • Canvas and SVG for graphics.
  • APIs like Geolocation, Web Storage, Web Workers, etc.
